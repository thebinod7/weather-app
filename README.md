# Weather Forcast

A Weather forecast app

## Run Locally

Setp 1: Clone the project

```bash
https://github.com/thebinod7/weather-app.git
```

Step 2: Go to the project directory and install dependencies

```bash
  cd weather-app
  yarn install
```

Step 3: Run project

```bash
   yarn start
```

Step 4: App is running at: http://localhost:3000
